# วัตถุประสงค์
ออกแบบ และพัฒนาระบบสนับสนุนการตัดสินใจสำหรับหัวหน้าหน่วยงานที่รับผิดชอบงานด้านสาธารณภัย พื้นที่เทศบาลตำบลสามโคก เพื่อนำข้อมูลไปใช้บริหารจัดการเตือนภัยอุทกภัยให้แก่ประชาชนที่มีความเสี่ยงที่จะได้รับผลกระทบ ตัดสินใจในการจัดลำดับความช่วยเหลือ และบรรเทาเหตุเมื่อเกิดภาวะอุกทกภัย 

# กรอบการวิจัย
- พื้นที่ศึกษาเน้นบริเวณสองริ่มฝั่งแม่น้ำที่ได้รับผลกระทบจากอุทกภัย
- ระบบเตือนภัยก่อนเกิดอุทกภัยให้กับเกษตรกร เช่น ช่วยเหลือชาวนาให้เก็บเกี่ยวข้าวได้ทันก่อนเกิดน้ำท่วม
- พัฒนาฐานข้อมูลของประชาชนกลุ่มเปราะบาง เช่น ผู้สูงอายุที่ไม่สามารถช่วยเหลือตนเองได้, ผู้ทุพลภาพ สตรีตั้งครรภ์
- พัฒนาฐานข้อมูลพื้นที่กลุ่มเปราะบางและมีความอ่อนไหวต่ออุทกภัย เช่น โบราญสถาน สถานที่สำคัญทางประวัติศาสตร์
